1. Brain Tumor Segmentation (YOLOv8) – AI for MRI Tumor Detection

Aim: To develop an automated system for detecting and segmenting brain tumors from MRI images.

Objective: Assist radiologists in early diagnosis by accurately identifying tumor regions using AI.

Skills Used: Python, Deep Learning, Image Processing, Data Preprocessing, Model Evaluation

Software Used: Python, YOLOv8, TensorFlow, OpenCV, Matplotlib

Procedure:

  1. Collected and preprocessed MRI datasets (normalization, noise reduction).
  2. Implemented YOLOv8 for tumor detection.
  3. Applied data augmentation and fine-tuned the model in TensorFlow.
  4. Refined tumor segmentation boundaries using OpenCV post-processing.
  5. Evaluated performance using Dice Coefficient, Precision, and Recall.

Result:

  • Achieved high accuracy in tumor detection and segmentation.
  • Reduced manual annotation effort for clinicians.

2. Cardiac Time Intervals using SCG – Signal Processing for Cardiac Dysfunction Detection

Aim: To extract cardiac time intervals from SCG signals to detect potential cardiac dysfunction.

Objective: Provide a non-invasive method to monitor heart function and assist in early diagnosis.

Skills Used: Signal Processing, Feature Extraction, Data Analysis, Visualization

Software Used: Python (SciPy, NumPy, Matplotlib), MATLAB

Procedure:

  1. Collected SCG signals from subjects.
  2. Preprocessed signals using bandpass filtering.
  3. Extracted key cardiac intervals (pre-ejection period, LV ejection time).
  4. Analyzed patterns using Python and MATLAB.
  5. Visualized SCG waveforms and intervals using Matplotlib.

Result:

  • Successfully extracted and visualized cardiac time intervals.
  • Enabled differentiation between normal and abnormal cardiac patterns.
